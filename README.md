# Cryptocurrency React app
Simple cryptocurrency price tracker made with React :heart:
[Check it out!](https://johnleoclaudio.github.io/crypto-ticker-app/)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

```
Node
```

### Installing

3 simple steps to run the app:

```
clone or download
```
Then
```
npm install
```
Lastly,
```
# Please make sure port 3000 is available
npm start
```
